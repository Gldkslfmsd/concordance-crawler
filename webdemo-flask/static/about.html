About ConcordanceCrawler

How does ConcordanceCrawler work?

ConcordanceCrawler finds links on Bing.com search engine, visits them and extracts concordances from them. During that it filters documents in encodings different than specified by user, filters Non-English documents, splits text to sentences and filters sentences containing concordances. It uses morfological analysis, so it is able to find ocurrences of target word in any of its grammatical forms. Final list of concordances can be formatted as json or xml.

ConcordanceCrawler also solves a problem that Bing.com (and all search engines) are able to offer at most 1000 links for every keyword, which is too few. Therefore ConcordanceCrawler lets finding keywords as for example "sdtn look", "naxe look", "jzmw look" and similar combinations of bazword and target word. By this approach it gets sufficient number of different links to crawl concordances.

How does ConcordanceCrawler look like?

ConcordanceCrawler is a command-line application intended for Linux platform. It's created in Python programming language (so it should work also on other platforms). It's compatible with both Python 2 and 3.

What are ConcordanceCrawler's best features?

ConcordanceCrawler is robust, it's able to crawl millions of concordances and it can run for weeks and months continuously. It's also gentle, it stops if some error disabling crawling of concordances occurrs, e.g. on errors with network communication. The main goal of implementation was to download corpora of the best quality as possible.
